* Fork this branch
* Create 2 git remote as follow: "upstream" for this repository, and "origin" for your forked repository
* When create new pull request, push to "origin" and ask to merge to "master"
* When working on new feature, create new branch using: git checkout -b <branch name>
* When coding, follow Ruby style guide: https://github.com/styleguide/ruby
* Suggest to use postgresql database. Install postgresql by: brew install postgresql
